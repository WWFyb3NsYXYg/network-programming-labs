# Network Programming Labs  
## Мережне програмування — Робота з HTTP протоколом, запитами та публічними API  

### Опис  
Цей репозиторій містить лабораторні роботи з дисципліни **«Мережне програмування»**.

---

## Структура репозиторію

```
network-programming-labs/
  lab1-http-api/      # Лабораторна робота №1 — HTTP, API, Telegram
  lab2-flask-api/     # Лабораторна робота №2 — Flask, HTTP сервер, API НБУ
  lab3-rest-api-flask-sqlite/   # Лабораторна робота №3 — REST API, Flask, JWT, SQLite
  lab4-socket-programming/        # Лабораторна робота №4 — Socket API (TCP)

```

Кожна лабораторна робота містить:
- вихідні файли Python,  
- зображення у теці `pictures/`,  
- файл `README.md` із описом і звітом.
